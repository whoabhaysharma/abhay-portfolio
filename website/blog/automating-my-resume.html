How I Automated My Resume Updates (Because I'm Lazy)

Look, updating a resume is the worst. You fight with Word margins, export 15 different PDFs named "final_final_v2", and then lose the source file. I decided to overengineer a solution so I never have to deal with that again.

The "Overkill" Tech Stack

I used a bunch of fancy tools to solve a simple problem. Why? Because I can.

LaTeX (The Code)

Because why drag and drop when you can write cryptic commands like \section{Education} and feel like a hacker?

GitHub Actions (The Minion)

My free cloud intern. It wakes up, builds my resume, and goes back to sleep. Doesn't ask for coffee breaks.

Jules (The Brains)

My AI pal. I just rant about my new job, and it translates my rambling into professional resume-speak.

Google Agents ↗

TmpFiles (The Dropzone)

A place to dump the file for 60 minutes so I can grab it. Simple, disposable, effective.

Check Service ↗

How It Started

I’m not a LaTeX wizard. I did what any respectable engineer would do: I asked ChatGPT to "Make me look professional in LaTeX".

I copy-pasted that into Overleaf, fixed the weird typos, and saved it as resume.tex. Boom. Resume as Code. Version controlled career history.

The "Laziness" Workflow

Now, when I need to update my resume, I don't even open the file. I treat my career updates like a drive-thru order.

1. "Hey Jules, Fix This"

I message my AI agent: "Yo, I just learned Kubernetes and saved the company 50 bucks. Add that to my resume."

Jules nods (digitally), writes the complex LaTeX markup, and opens a Pull Request.

2. The Cloud Factory

I hit "Merge" on my phone. GitHub Actions spins up a server, installs a massive LaTeX distribution (sorry GitHub bandwidth), and compiles the document.

It creates a pixel-perfect PDF while I'm probably making a sandwich.

3. Special Delivery

The robot finishes up, uploads the file to tmpfiles.org, and slides into my Slack DMs.

  • Ding! "New Resume Ready".
  • I click the link.
  • I send it to the recruiter.

Was It Worth It?

I spent 10 hours automating a task that takes 10 minutes. So, obviously, yes.

But seriously, having my resume as code means no more formatting nightmares. And it just feels cool.
