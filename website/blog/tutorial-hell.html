Tutorial Hell is Warm and Cozy

// DATE: 2025-12-20
// MOOD: CAFFEINATED
// TAGS: #RUST #PAIN

I watched a 4-hour tutorial on Rust. I now know how to print "Hello World" in a memory-safe way.

I will never use this skill again.

The Cycle

  1. Get excited about new tech.
  2. Watch "Build X in 10 minutes".
  3. Follow along perfectly.
  4. Try to add one feature.
  5. Everything breaks.
  6. Go back to Step 1 with different tech.
I have built 14 To-Do apps. I still forget to buy milk.

Escaping Hell

The only way to learn is to build something that breaks. Not something from a video where the instructor has already fixed all the bugs before hitting record.

So I decided to build this portfolio. It's ugly, it's brittle, and the code is a mess. But hey, at least I didn't need a tutorial to center the div. (Okay, maybe I did).
